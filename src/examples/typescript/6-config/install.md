1. Установка TypeScript

глобально для всех проектов
```bash
npm i -g typescript
```
или локально для конкретного проекта
```bash
npm i -D typescript
```


2. TypeScript REPL
```bash
npm i -g ts-node
```

3. Запуск компилятора
```bash
tsc --help
```

4. Запуск .ts файла без компиляции
```bash
ts-node index.ts
```
